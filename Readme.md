# Functional Wrapper for OAuth2.0

## Current Version
0.3.1

## Tested and working with following oauth Endpoints
* Hubspot 
* facebook
* Google

## Python Version
3.7.2

## Installation
For installing execute the following.
```python 
pip install git+https://github.com/troopertino/oauth2py.git
```
